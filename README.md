# ExploradorFTPDataSUS

**Explore, Navegue e Analise Dados de Saúde Pública do Servidor FTP do Governo Brasileiro com Facilidade!**

## Visão Geral

O ExploradorFTPDataSUS é uma ferramenta de interface de linha de comando (CLI) projetada para simplificar a exploração, navegação e recuperação de dados de saúde pública do servidor FTP do governo brasileiro. Seja você um desenvolvedor, analista de dados ou pesquisador, essa ferramenta facilita o processo de acesso e análise de informações valiosas de saúde.

## Recursos

- **Navegação sem Esforço:** Navegue facilmente pela estrutura de diretórios do servidor FTP com um menu intuitivo, sem necessidade de linhas de código diretamente.

- **Download de Arquivos:** Baixe arquivos específicos, tornando a recuperação de dados simples.

- **Funcionalidade de Busca:** Procure rapidamente por arquivos com base em palavras-chave ou critérios, economizando tempo na localização dos dados desejados.

## Em Breve

- **Utilização Simplificada:** Em breve será implementada uma UI para facilitar a utilização do software por usuários menos versados em linhas de comando ou terminais.

## Começando

0. Instale o GIT: [https://git-scm.com/downloads](https://git-scm.com/downloads)
1. Abra o terminal
2. Clone o repositório: `git clone https://github.com/gabrielfmcoelho/ExploradorFTPDataSUS.git`
3. Vá para a pasta: `cd ExploradorFTPDataSUS`
4. Instale as dependências: `pip install -r requirements.txt`
5. Execute a CLI: `python app.py`

Para executar o programa em outro momento, basta abrir o terminal novamente, navegar até a pasta com o comando do 3º passo e, por fim, executar a CLI com o comando do 6º passo.

## Exemplos de Uso

- Listar diretório: `1`
- Navegar até um diretório: `2` -> `nome_do_diretorio`
- Voltar: `3`
- Baixar um arquivo: `4` -> `nome_do_arquivo.csv`
- Procurar por arquivos: `5` ou `6` -> `palavra-chave` ou `data`
- Sair: `0`
